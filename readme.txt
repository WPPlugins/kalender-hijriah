== Description ==

Display a hijri flash calendar on your sidebar.
(Menampilkan flash kalender hijriah di sidebar.)

== Installation ==

1. Download the zip file and extract the contents (download file zip & ekstrak isinya),
2. Upload the folder `kalender-hijriah` to your WP plugin folder `/wp-content/plugins/` directory (upload folder `kalender-hijriah` & isinya ke folder WP plugin `/wp-content/plugins/`),
3. Go to Plugins > Installed, and activate the plugin (Pindah ke menu Plugins > Installed, & aktifkan plugin),
4. Go to Appearance > Widgets, and drag the widget to a sidebar (Pindah ke menu Appearance > Widgets, & aktifkan plugin),
